# L'atelier des Gourmets — Next.js + React (accessible, SEO)

- Mobile-first, responsive (CSS grid + modern viewport units).
- Navigation accessible (skip-link, landmarks, aria-current).
- Headings hiérarchisés, labels explicites, focus visible.
